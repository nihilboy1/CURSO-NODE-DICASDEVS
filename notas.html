// O node e o Chrome usam a mesma engine de processamento. Engine V8 // O node é "siglethreded" e "non-blocking": O node é como um garçom que anota um pedido, entrega na cozinha e ja vai pra próxima mesa. Quando o pedido da primeira mesa ficar pronto, ele volta na cozinha, pega o pedido e entrega na mesa. E assim sucessivamente // módulos buildt-in do node: path, fs, http // NPM (node package manager) "node_modules" armazena todos os módulos instalados "package.json" lista todos os módulos instalados pelo NPM "npm init -y" inicia o npm e cria o package.json "npm install" instala um pacote localmente "npm -g install" instala um pacote globalmente
